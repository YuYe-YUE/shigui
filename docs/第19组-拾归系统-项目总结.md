> 随开发进度持续更新。当前版本：Sprint 8 完成（2026-05-18）。

## 1. 项目信息

- **项目名称**：拾归 — 校园失物招领系统
- **开发团队**：第19组（7人）
- **技术栈**：Spring Boot + MyBatis-Plus + Sa-Token（后端）、Vue 3 + Element Plus（管理端）、原生微信小程序（用户端）、MySQL 8.0

## 2. 开发方法

## 4. 技术选型理由

- **Spring Boot 3.5**：Java 生态最成熟的 Web 框架，课程教学全覆盖
- **MyBatis-Plus**：Lambda 查询 + 分页插件 + 逻辑删除，开发效率高于原生 MyBatis
- **Sa-Token**：轻量级认证框架，注解声明鉴权，无需 Spring Security 笨重配置
- **Vue 3 + Element Plus**：组合式 API + 丰富组件库，管理端快速搭建
- **原生小程序**：避免 uni-app/taro 额外学习成本，团队可直接上手
- **SHA-256 + salt 密码**：不依赖 BCrypt 库，减少依赖复杂度

## 5. 遇到的问题与解决

### 5.1 Maven Central 网络不通
`spring-boot:run` 的插件运行时依赖无法下载。**解决**：用 IDEA 直接运行 `BackendApplication.java`，绕过 Maven 插件。

### 5.2 Sa-Token 排除路径与 POST 冲突
`/api/posts` 同时有公开列表（GET）和发布（POST），Sa-Token 按路径排除不区分 HTTP 方法。**解决**：`/api/posts/**` 全排除，POST/mine 手动验登录。

### 5.3 MyBatis-Plus + Mockito lambdaQuery 兼容
`ServiceImpl.lambdaQuery()` 依赖 Mapper 代理对象元数据，Mockito mock 缺少这些数据导致异常。**解决**：反射注入 `entityClass`/`mapperClass`。

### 5.4 微信 tabBar SVG 不支持
**解决**：Python 纯手写 PNG 编码，生成 81x81 RGBA 图标。

### 5.5 管理员接口无专属鉴权
**解决**：管理员 ID 偏移 10,000,000，`requireAdmin()` 校验，普通用户→403。

### 5.6 审核操作非事务
**解决**：抽取 `AdminPostService` + `@Transactional`。

### 5.7 详情接口泄露待审核单据
**解决**：`getDetail` 加 currentUserId 参数，非本人只能看 MATCHING。

### 5.8 图片 URL 相对路径
后端返回 `/uploads/...` 相对路径，小程序 `<image>` 无法加载。**解决**：`app.resolveImageUrl()` 给相对路径加 baseUrl。

### 5.9 Spring 多构造器无法实例化
`LocalFileStorageServiceImpl` 两个构造器，Spring 不知道用哪个。**解决**：加 `@Autowired` 标注。
