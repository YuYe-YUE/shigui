## 1. 测试策略

三层测试：Service 单元测试（JUnit 5 + Mockito）、Controller 集成测试（@SpringBootTest + MockMvc）、Mapper 测试（MyBatis-Plus 内置）。每个 Sprint TDD 开发——先写测试确认失败，再实现代码验证通过。

```bash
cd backend && ./mvnw test
```
